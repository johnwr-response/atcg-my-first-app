###### Updating NPM
```
npm install -g npm
```
###### Updating CLI
```
npm uninstall -g angular-cli @angular/cli
npm cache verify
npm install -g @angular/cli
```

###### Run on different port
```
ng serve --port xxxx
```

###### install bootstrap (This course relies on bootstrap 3)
```
npm install --save bootstrap@3
```

git add .
